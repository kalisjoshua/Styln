Styln - CSS for form layout

Styln is focused on the layout of properly marked up html forms, therefore it only affects the form elements position and layout. Styln only attempts to baseline and enhance some aspects of the form elements. Some features not effected by Styln are

  • font-size - besides where necessary to unify conformity
  • grid system for elements outside the form
